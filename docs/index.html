An accessible English on-screen keyboard built for real work

LAltKey helps you type, trigger shortcuts, and manage repeated text with a mouse, touch screen, external switch, or keyboard navigation. It keeps the mature accessibility and tooling foundation from AltKey while keeping the input path focused on English.

FAQ

Common questions

These answers focus on the English-only LAltKey edition.

How is LAltKey different from AltKey?

LAltKey is the English-only sibling project. It keeps accessibility, editor tools, action keys, and prediction, but narrows the input system and layouts to English.

Can it be installed alongside AltKey?

Yes. LAltKey uses its own executable names, app data folder, reload events, startup entry, installer names, and update endpoint.

What does word prediction learn?

It records completed or accepted English words and learns word pairs so the next suggestion can reflect recent context.

Why are editor tools separate from the keyboard?

The tools process lets you edit layouts, dictionaries, shortcuts, profiles, and prompts without mixing heavy editing workflows into the no-activate keyboard window.

Can it type into elevated apps?

Windows may block input from a normal-permission app into an administrator-permission app. LAltKey shows warnings and can be restarted with matching permissions when needed.
